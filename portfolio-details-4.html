Interactive widget for embedding in streams and database collecting results of previous games

In this project, with the help of a backender colleague, i designed, developed and launched a special project for casino streamers. The task was to create a widget placed on a background that is easy to chroma key.

  • The application calculates statistics, and the widget displays the results of the current game session and the streamer's game history in the past.
  • The user can track his progress over time, share this data with the audience.
  • The widget scales proportionately to user-specified sizes and looks great when grabbing the screen.


    Environment: VScode, GIT(GitLab), JS, HTML, CSS, LAMP, VueJS 2, vuex, axios,
